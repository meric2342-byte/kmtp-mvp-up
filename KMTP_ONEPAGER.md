# **🏥 KMTP — 한국 의료관광 신뢰 인프라 플랫폼**

**KMTP**는 외국인 환자가 한국에서 의료 서비스를 받는 전 과정을 **신뢰할 수 있게 연결**하는 의료관광 운영 플랫폼입니다. 병원을 단순히 소개하는 곳이 아니라, 예약 시점에 잠근 가격이 입국 후에도 그대로 청구되는 **가격잠금**, 결제금을 안전하게 보관하는 **에스크로**, 입국부터 출국까지의 **실시간 여정 추적**까지 — 환자·유치업자(에이전트)·병원을 하나의 신뢰 체계로 묶습니다.

> 병원을 소개하는 곳이 아니라, 외국인 환자의 한국 의료 여정을 신뢰 가능하게 연결하는 신뢰 운영체계.

## **✨ 핵심 기능**

### **1. 3역할 신뢰 운영 (환자 · 에이전트 · 병원)**

하나의 플랫폼에서 역할에 따라 다른 화면을 제공하고, 민감 정보를 **역할별로 격리**합니다. 환자는 견적·여정을, 에이전트와 병원은 담당 환자 현황만 보며 **견적·수수료는 노출되지 않습니다.** 환자는 직접 회원가입할 수 있고, 에이전트·병원 계정은 **관리자(admin)** 가 발급합니다.

- **사용 예시:** *환자는 "내 여정"에서 진행 상황을, 에이전트는 담당 환자 픽업 현황을, 병원은 예약 환자 도착 상태를 각각 확인.*

### **2. 가격잠금 · No-Surprise · 에스크로 (신뢰 3모듈)**

진료과에 따라 **가격잠금 3종**(총액 고정 / 범위 보장 / 상담 견적)을 적용하고, 입국 후 실제 청구서가 예약 시점 견적과 **일치함을 눈으로 증명**합니다. 결제금은 병원으로 바로 가지 않고 치료 완료 확인 시까지 **에스크로**에 안전 보관됩니다.

- 🔒 **가격잠금:** 예약 시점 가격을 잠가 입국 후에도 그대로 (환율 ±3% 자동 흡수)
- ✅ **No-Surprise 증명:** 잠근 견적 = 실제 청구서 좌우 비교, "추가 청구 없음"
- 🛡️ **Trust Score 5축:** 가격잠금 준수율·슬롯 성실도·결과 응답률·만족도·합병증율 — 광고로 살 수 없는, 실거래 데이터 기반 점수

### **3. 실시간 여정 추적 + 상황별 카카오톡 스타일 알림**

입국부터 출국까지 **9단계 여정**(현지 출발 → 공항 도착 → 픽업 → 회복스테이 → 병원 → 시술 → 회복 → 재진 → 출국)을 실시간 타임라인으로 보여주고, 각 단계·픽업 상황마다 **카카오톡 스타일 말풍선 메시지**로 안내합니다. 환자는 픽업 기사에게 직접 메시지를 보낼 수도 있습니다.

- **사용 예시:** *"🚐 공항 픽업이 완료되었습니다. 숙소로 이동합니다." 메시지가 환자·에이전트·병원에게 자동 전송, 회복기간에 맞춘 회복스테이 객실도 자동 매칭.*
